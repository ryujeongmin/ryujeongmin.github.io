PROJECT

Tangled Emotion

A journaling interface that helps users externalize complex feelings into structured “threads” — extracting emotion nodes, identifying key knots, and generating actionable reflections through an AI diary companion.

Core Experience

Users write freely without format constraints. The system then “untangles” the writing into emotion nodes and a causal thread, highlighting recurring knots and decision points.

My Records

A split-layout journal view: a lightweight writing prompt on the left, and a structured log on the right. Each entry is archived with a title, date, tags, and an AI-generated response to preserve continuity over time.

Persona Settings

Users can select the AI companion’s tone (e.g., warm friend / realistic advisor / playful muse). This allows the same emotional structure to be reflected back in a voice that matches the user’s preference, without changing the underlying analysis logic.

Profile Memory

A user-editable profile summary supports personalization. The system uses this context to reduce repetitive explanations, maintain consistency in feedback, and detect longer-term emotional patterns across entries.
