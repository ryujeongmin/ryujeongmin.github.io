PROJECT

API 변동성 기반 자동매매 시스템

한국투자증권 API를 이용하여 변동성을 고려한 자동매매 시스템을 구현합니다.
실시간 주가 데이터를 분석하여 매수/매도 타이밍을 자동으로 결정하는 알고리즘 트레이딩 프로젝트입니다.

알고리즘 원리

본 시스템은 주가의 변동폭(Volatility)을 계산하여 안전한 진입 시점을 포착합니다.

주요 용어:
- 변동폭 (P_delta): 전일 고가와 저가의 차이 (High - Low)
- 매수 기준가 (Target Price): 당일 시가 + (변동폭 * k)
- 매도 기준가: 목표 수익률 도달 시 자동 매도

Implementation

Python과 한국투자증권 Open API를 연동하여 실시간 시세 조회 및 주문 실행을 자동화하였습니다.
비동기 처리(AsyncIO)를 통해 다수의 종목을 동시에 모니터링하며, 텔레그램 봇 연동을 통해 매매 체결 내역을 실시간으로 알림받습니다.
