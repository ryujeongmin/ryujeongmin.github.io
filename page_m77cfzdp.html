PROJECT

Robot Arm with Self Driving Mobility

Combined a self-driving mobile base with a robotic arm.
The system integrates autonomous navigation with manipulation capabilities.

Mechanism

Based on the kinematics of the robot arm, we derived the DH parameters and found the transformation matrix. We utilized Python libraries to implement the inverse kinematics code, allowing precise control of the arm's movement.
