Technical Architecture

Stack & system design

Built as a full-stack web app on Firebase Cloud Workstation. The architecture prioritizes persistent user context 芒聙聰 the system maintains a live profile JSON that is updated after every entry using a secondary GPT-3.5 call to extract profile updates from the new diary text.

01
Input & GPT-4 response 芒聙聰 User writes free text. A curated system prompt containing the full user profile and recent diary history is sent to GPT-4, which returns a structured JSON with title, date, type, content, and diary feedback.
02
Profile update loop 芒聙聰 A second GPT-3.5 call extracts any new information from the entry and merges it into the persistent user profile, making every future response more accurate.
03
Firebase persistence 芒聙聰 Entries and profile are stored in Firestore with Firebase Auth for secure single-user access. The diary list updates in real-time.
04
Persona injection 芒聙聰 The selected persona modifies the system prompt's emotional register, changing the tone of AI feedback without altering the underlying data structure.
