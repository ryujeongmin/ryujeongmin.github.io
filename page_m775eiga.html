PROJECT

FoldiFolda

A chair-mountable robotic avatar system tailored for activity-based group learnings in hybrid educational settings. The system consists of a controller (Foldi), which serves as both a laptop stand and a control interface, and a chair-mountable robot avatar (Folda).

Demo Video

Demonstrating the FoldiFolda system in action for remote group activities. The robot mimics the remote user's head movements and gestures, creating a sense of presence.

Control UI

The control interface allows the remote user to guide the robot's gaze and gestures directly from their laptop. It serves as an intuitive bridge between the physical and digital workspaces.

Installation Step

FoldiFolda is designed for easy installation on standard chairs. The modular design ensures it can be quickly set up in any classroom or meeting room environment without complex tools.

Software Architecture

The system integrates real-time communication protocols to sync the robot's physical movements with the user's commands, ensuring low-latency interaction for natural group dynamics.

Hardware Exploded View

A detailed look at the mechanical structure, including the servos, joints, and mounting mechanisms that allow Folda to move expressively while remaining stable on a chair back.
