Image Perfume

Translating photographs into fragrance — cross-modal memory made physically tangible.

InstitutionKAIST
TypeHCI Research
Year2023
CoverageHelloDD News
Overview

RGB → scent composition

Image Perfume converts an image into a unique perfume. By analyzing the RGB composition of a photograph, the system creates a scent profile layered to match the visual memory. Each R, G, B channel corresponds to a specific base scent; the machine layers these according to the ratio of values found in the image.
Image Perfume — the scent extraction device
Process

How it works

The interaction is designed to feel like a ritual — extracting the essence of a memory and making it breathable.
01
Place the photograph in the designated slot.
02
Press the button — the machine analyzes the image's color composition.
03
Wait for the RGB-to-scent mapping to complete.
04
Receive the dispensed scent profile unique to that image.
Design Intent

Synesthetic memory

The project explores how sensory translation can strengthen memory. Looking at a photo activates visual recall; its corresponding scent activates olfactory recall. Together they create a richer, multi-channel memory experience. Featured in HelloDD News as a novel approach to cross-modal HCI.
